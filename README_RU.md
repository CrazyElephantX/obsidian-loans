# Obsidian Loans Tracker

Плагин для Obsidian, предназначенный для учета кредитов, отслеживания платежей и расчета выгодного досрочного погашения.

## Возможности

- 📊 **Учет кредитов**: Добавление, редактирование и удаление кредитов с детальными параметрами (сумма, процентная ставка, срок, тип платежа).
- 📅 **График платежей**: Автоматическое создание графика платежей (аннуитетных или дифференцированных).
- ✅ **Отслеживание платежей**: Отметка выполненных платежей, просмотр статуса (ожидает, оплачен, просрочен).
- 📈 **Статистика**: Визуализация прогресса погашения, остатка долга, общей переплаты.
- 🧮 **Калькулятор выгодного погашения**: Расчет, какой кредит выгоднее погасить при наличии свободных средств.
- 🔔 **Уведомления**: Автоматические напоминания о просроченных платежах.
- ⚙️ **Настройки**: Гибкая конфигурация валюты, файла хранения данных и уведомлений.
- 🌐 **Поддержка языков**: Автоматическое определение языка Obsidian (русский/английский).

## Установка

### Ручная установка
1. Скачайте последний релиз (или клонируйте репозиторий).
2. Скопируйте папку плагина в `.obsidian/plugins/obsidian-loans/` вашего хранилища Obsidian.
3. Включите плагин в настройках Obsidian (Community plugins).

### Установка через BRAT (рекомендуется)
1. Установите плагин [BRAT](https://github.com/TfTHacker/obsidian42-brat).
2. Добавьте репозиторий: `https://github.com/ваш-username/obsidian-loans`.
3. Включите плагин в настройках.

## Использование

### Добавление кредита
1. Нажмите на иконку `$` в боковой панели или выполните команду `Loans Tracker: Открыть панель учета кредитов`.
2. Нажмите кнопку `➕ Добавить кредит`.
3. Заполните форму:
   - **Название кредита**: Произвольное имя для идентификации.
   - **Сумма кредита**: Общая сумма займа.
   - **Процентная ставка**: Годовая процентная ставка.
   - **Срок**: Количество месяцев.
   - **Дата начала**: Дата выдачи кредита.
   - **Тип платежа**: Аннуитетный (равные платежи) или дифференцированный (уменьшающиеся).
   - **Валюта**: RUB, USD, EUR и т.д.
   - **Заметки**: Дополнительная информация.
4. Нажмите "Сохранить". Плагин автоматически сгенерирует график платежей.

### Просмотр кредитов
В главном окне отображается:
- Общая статистика: количество кредитов, общий долг, выплаченная сумма, переплата.
- Список кредитов с прогресс-баром погашения.
- Для каждого кредита показаны остаток долга и процент погашения.

### Отметка платежа
1. В главном окне нажмите кнопку "Платежи" для нужного кредита.
2. В таблице платежей нажмите "Отметить оплату" для ожидающего платежа.
3. Укажите фактически уплаченную сумму и дату оплаты.

### Калькулятор выгодного погашения
1. Нажмите кнопку `🧮 Калькулятор погашения` в главном окне.
2. Введите сумму, которую можете направить на досрочное погашение.
3. Плагин предложит оптимальную стратегию: погасить кредит с наибольшей процентной ставкой для максимальной экономии.

## Настройки

Перейдите в `Настройки Obsidian → Плагины → Loans Tracker`:

- **Файл хранения данных**: Путь к JSON-файлу, где хранятся данные (по умолчанию `loans-data.json`).
- **Валюта по умолчанию**: Валюта для новых кредитов.
- **Уведомления**: Включить/отключить уведомления о просроченных платежах.
- **Авторасчет платежей**: Автоматически пересчитывать график при изменении параметров кредита.

- Node.js 16+

```bash
npm install
```
